# jusan-cli

Это cli проект. 

## Возможности

Данный проект имеет следующий функционал.

- Запрашивает запросы через **stdin**.
- Выводит в **stdout** уникальный идентификатор.
- Основные команды записаны в help.txt.

## Запуск

Для компиляций проекта.

``` 
javac Myfiles.java
```

Для запуска программы.

``` 
javac Myfiles.java
```
